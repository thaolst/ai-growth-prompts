Các prompt về retention strategy: thiết kế retention loop, Xu economy, churn prediction và re-engagement campaign.

Xem [`SKILL.md`](./SKILL.md) cho nội dung đầy đủ.

## 🎯 Ví dụ output

**Đầu vào:** App muốn giảm churn 15% trong 3 tháng, user base 2M MAU, Xu economy đã có.

**Kết quả:**

> **Retention loop đề xuất:**
> * Week 1: Xu sign-up bonus → tạo habit
> * Week 2-4: Daily streak reward → giữ frequency
> * Month 2: Tier unlock (Silver/Gold) → switching cost
> * Month 3: Referral loop → viral retention
>
> **KPI tracking:**
> * D1 retention: baseline 45% → target 55%
> * D7 retention: baseline 20% → target 30%
> * D30 retention: baseline 8% → target 15%
