Full content + prompt → `SKILL.md`

### 🎯 Example output

**Input:** Campaign T12, voucher budget 500M VND, target 200K redemptions, 2 types: 50K flat discount (new user) and 20% cashback (existing).

**Result:**

> **Budget split:**
> * New user (50K flat): 300M, 6K vouchers, target 150K redemptions
> * Existing (20% cashback): 200M, unlimited, target 50K redemptions
>
> **Expected ROI:**
> * New user: 30% redemption → 45K new users, CPA ~6.6K
> * Existing: 25% redemption → 12.5K returning users
>
> **Risks to monitor:**
> * New user: multi-account fraud
> * Existing: cashback expiry before use
